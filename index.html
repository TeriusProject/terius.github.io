Product
Terius si propone principalmente di sviluppare un prodotto composto da due
TDB ‐ Terius DataBase

Un Database distribuito che possa contenere il maggior numero possibile di informazioni agricole affidabili, testate, e formattate in un formato standard (ADF). Questo database deve poter essere iterrogato sia tramite API pubbliche, che tramite un'apposita interfaccia web, che possa visualizzare i dati in maniera interattiva tramite dashboard di grafici e tabelle. Integrato con il database deve essere presente anche un servizio di reportistica.

Market placement
Terius AI Engine

Un modello cloud on-demand che permette di

  • Automatizzare completamente serre e coltivazioni
  • Fare simulazioni per testare la crescita di una coltura in un ambiente arbitrario
  • Determinare quali siano le condizioni ambientali ottimali per una data coltura, sualla base delle analisi condotte sui dati immagazzinati in Terius Database
